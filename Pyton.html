01 Cos'è Python e la differenza con C++

Python è un linguaggio "amichevole". È progettato per permetterti di scrivere programmi velocemente, concentrandoti sull'idea e non su regole tecniche troppo complicate.

Caratteristica Python C++
Semplicità Molto alta. Si scrive poco codice. Più complesso. Serve molto codice.
Rigidità Flessibile: "capisce" da solo molte cose. Rigido: devi spiegare ogni minimo dettaglio al computer.
Velocità Leggermente più lento nell'esecuzione. Velocissimo, usato per videogiochi pesanti.
02 Dichiarazione delle Variabili

Immagina una variabile come una scatola con un'etichetta sopra. Dentro la scatola puoi mettere un dato (un numero, una parola).

In Python, non devi dire al computer che tipo di scatola vuoi; basta darle un nome e metterci dentro qualcosa.

Variabile Numerica

Serve per fare calcoli. Non servono virgolette.

  • Esempio: mele = 5
  • Abbiamo creato una scatola chiamata mele e ci abbiamo messo il numero 5.
Variabile Stringa

La "stringa" è una sequenza di caratteri (testo). Per far capire a Python che è testo e non un comando, si usano le virgolette " ".

  • Esempio: nome_frutto = "Mela Rossa"
  • Abbiamo messo il testo "Mela Rossa" nella scatola nome_frutto.
03 L'Assegnazione: Cos'è e come si fa?

Assegnare significa letteralmente "mettere un valore dentro una variabile". Si usa il simbolo uguale =.

Assegnazione Numerica:

prezzo = 10
Cosa succede? Il computer riserva uno spazio e memorizza il numero 10.

Assegnazione Stringa:

saluto = "Ciao a tutti!"
Cosa succede? Il computer memorizza la frase. Senza le virgolette, Python cercherebbe di capire se Ciao è un comando, andando in errore.
04 La Condizionale: Cos'è?

Una condizionale è un bivio. Permette al programma di prendere decisioni: "Se succede questo, fai così; altrimenti, fai cosà". In Python si usa la parola if (che in inglese significa "se").

⚠ Regola fondamentale: l'indentazione in Python
Python è molto preciso con gli spazi. A differenza di altri linguaggi, gli spazi non sono decorativi: sono parte della grammatica del codice.
  • Dopo if ...: e else: il codice interno deve essere rientrato di 4 spazi (o 1 Tab).
  • Se dimentichi i 4 spazi, Python va in errore e il programma non parte.
  • Tutti i comandi allo stesso livello devono avere lo stesso rientro.
