SCHEDA 1 — Cos'è una potenza? Scheda di matematica · Prof. Belardi Nicolino
Una potenza vuol dire: moltiplicare lo stesso numero tante volte
3⁵ = 3 × 3 × 3 × 3 × 3
🔵
3 è la BASE → il numero in basso (grande) che moltiplico
5 è l'ESPONENTE → il numerino in alto che dice quante volte
💡
BASE = numero grande in basso · ESPONENTE = numerino piccolo in alto
📏
SCHEDA 2 — Regole del segno Scheda di matematica · Prof. Belardi Nicolino
Il risultato di una potenza dipende dall'esponente.
✅
Esponente PARI (2, 4, 6…) → risultato sempre POSITIVO (+)
Es: (−3)² = 9 → positivo!
⚠️
Esponente DISPARI (1, 3, 5…) → stesso SEGNO della base
Es: (−3)³ = −27 → negativo come la base
0️⃣
Base = 0 → risultato sempre 0
Es: 0⁵ = 0
🔑
Pari → sempre +  ·  Dispari → stesso segno della base
📐
SCHEDA 3 — Le regole delle potenze Scheda di matematica · Prof. Belardi Nicolino
✖️
Moltiplico, stessa base → sommo gli esponenti
5⁴ × 5³ = 5⁷
➗
Divido, stessa base → sottraggo gli esponenti
9⁴ ÷ 9² = 9²
🔁
Potenza di potenza → moltiplico gli esponenti
(3³)² = 3⁶ = 729
1️⃣
Esponente = 0 → risultato sempre 1
9⁰ = 1 · qualsiasi⁰ = 1
🔄
Esponente negativo → metto 1 sopra (frazione)
10⁻² = 1/10² = 1/100
💡
× stessa base → somma · ÷ stessa base → sottrai · esponente 0 → 1 · esponente − → vai sotto
🔵
SCHEDA 4 — Esercizio riepilogativo Scheda di matematica · Prof. Belardi Nicolino
(ax)² · a3x ÷ a · 1/a · ³√a²
1
Potenza di potenza
(ax)² → a2x x × 2 = 2x
↓
2
Moltiplico per a3x
a2x · a3x → a5x 2x + 3x = 5x
↓
3
Divido per a
a5x ÷ a → a5x · a−1 → a5x−1
↓
4
Moltiplico per 1/a
a5x−1 · a−1 → a5x−2 (5x−1)+(−1)
↓
5
Trasformo la radice cubica
³√a² → a2/3 indice 3 sotto, esponente 2 sopra
↓
6
Moltiplico l'ultimo fattore
a5x−2 · a2/3 → −2 = −6/3
−6/3 + 2/3 = −4/3 → a5x−4/3
🏆 RISULTATO FINALE
a5x − 4/3

SCHEDA 5 — La funzione esponenziale Scheda di matematica · Prof. Belardi Nicolino
y = bx

b = base (numero fisso, positivo, ≠ 1)
x = esponente che cambia ogni volta
✅ Passa sempre per (0 , 1) perché b⁰ = 1
✅ L'asse x è un asintoto orizzontale → la curva si avvicina ma non tocca mai y = 0
✅ y è sempre positiva, mai zero o negativa
💡
Asintoto = linea che la curva insegue ma non raggiunge mai, come l'orizzonte.
⚠️
y = 2x → esponenziale (esponente = x, base fissa)
y = x² → polinomiale (base = x, esponente fisso)
📊
