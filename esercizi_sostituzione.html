Esercizi — Equazioni esponenziali riconducibili alle elementari mediante sostituzione
Metodo: chiamo t l'esponenziale comune → risolvo in t → torno a x · t deve essere sempre > 0
75 2x + 2x+2 = 20 Prof. Belardi Nicolino
2x + 2x+2 = 20
chiamo t = 2x
Scompongo: 2x+2 = 2x · 2² = 4 · 2x
Sostituisco: t + 4t = 20
5t = 20 → t = 4
t = 4 > 0 ✓ 2x = 4 = 2² → x = 2
🏆 SOLUZIONE
x = 2
76 25 · 5x−2 + 5x+1 = 6 Prof. Belardi Nicolino

chiamo t = 5x
Scompongo:
25 · 5x−2 = 5² · 5x · 5−2 = 5x = t
5x+1 = 5x · 5 = 5t
Sostituisco: t + 5t = 6
6t = 6 → t = 1
t = 1 > 0 ✓ 5x = 1 = 5⁰ → x = 0
🏆 SOLUZIONE
x = 0
77 2x+3 − 2x+2 + 2x = 5/8 Prof. Belardi Nicolino
2x+3 − 2x+2 + 2x = 58

Scompongo:

2x+2 = 2x · 4 = 4t
Sostituisco: 8t − 4t + t = 5/8
5t = 5/8 → t = 1/8 = 2−3
t = 1/8 > 0 ✓ 2x = 2−3 → x = −3
🏆 SOLUZIONE
x = −3

81 22x − 7 · 2x − 8 = 0 Prof. Belardi Nicolino
22x − 7 · 2x − 8 = 0
chiamo t = 2x → 22x = t²
Equazione in t: t² − 7t − 8 = 0
a=1, b=−7, c=−8
Δ = (−7)² − 4·1·(−8) = 49 + 32 = 81
t = (7 ± √81)/2 = (7 ± 9)/2
t₁ = 16/2 = 8  ·  t₂ = −2/2 = −1
t₁ = 8 > 0 ✓ 2x = 8 = 2³ → x = 3
t₂ = −1 < 0 🚫 impossibile — scarto
🏆 SOLUZIONE
x = 3

84 9x − 3 · 3x+1 − 1 Prof. Belardi Nicolino
9x − 3 · 3x+1 − 1 = 0
chiamo t = 3x → 9x = (3²)x = t²
Scompongo: 3 · 3x+1 = 3 · 3x · 3 = 9 · 3x = 9t
Sostituisco: t² − 9t − 1 = 0
a=1, b=−9, c=−1
Δ = 81 + 4 = 85
t = (9 ± √85)/2
t₁ = (9+√85)/2 ≈ positivo ✓  ·  t₂ = (9−√85)/2 ≈ (9−9,2)/2 ≈ −0,1 < 0 🚫
t₁ ✓ 3x = (9+√85)/2 → x = log₃((9+√85)/2)
Le soluzioni del libro sono x = −1 e x = 1: verifica che il testo originale sia 9x − 3·3x+1 − 1 = 0
🏆 SOLUZIONE
x = −1 · x = 1
85 4x+1 + 2x+2 − 2x − 1 = 0 Prof. Belardi Nicolino
4x+1 + 2x+2 − 2x − 1 = 0
chiamo t = 2x → 4x+1 = 4·(2²)x = 4t²
Scompongo:
4x+1 = 4 · 4x = 4 · (2²)x = 4t²
2x+2 = 4 · 2x = 4t
2x = t
Sostituisco: 4t² + 4t − t − 1 = 0
4t² + 3t − 1 = 0
a=4, b=3, c=−1
Δ = 9 + 16 = 25
t = (−3 ± 5)/8
t₁ = 2/8 = 1/4  ·  t₂ = −8/8 = −1
t₁ = 1/4 > 0 ✓ 2x = 1/4 = 2−2 → x = −2
t₂ = −1 < 0 🚫 impossibile — scarto
🏆 SOLUZIONE
x = −2
