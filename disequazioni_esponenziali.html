SCHEDA — Disequazioni esponenziali Scheda di matematica · Prof. Belardi Nicolino
📖 1 — Definizione
Una disequazione esponenziale è una disequazione dove l'incognita x compare all'esponente.
La forma a cui vogliamo sempre arrivare è:
af(x) > ag(x)
(il simbolo > può essere <, ≥, ≤)
💡 Una volta ricondotta a questa forma, elimino le basi e lavoro solo sugli esponenti.
Attenzione: il verso della disequazione può cambiare in base al valore della base!
⚖️ 2 — La regola fondamentale: il ruolo della base
📈 Base > 1 — funzione CRESCENTE
a > 1 → verso INVARIATO
af(x) > ag(x)
⟹ f(x) > g(x)

Esempio: 2x > 2³
⟹ x > 3
Soluzione: x ∈ (3, +∞)
📉 0 < Base < 1 — funzione DECRESCENTE
0<a<1 → verso INVERTITO
af(x) > ag(x)
⟹ f(x) < g(x)

Esempio: (1/2)x > (1/2)³
⟹ x < 3
Soluzione: x ∈ (−∞, 3)
📏 Come scrivere la soluzione — notazione con intervalli
Estremo ESCLUSO → parentesi tonda ( )
Simboli: > oppure <
Il pallino sulla retta è vuoto ○
Es: x > 3 → x ∈ (3, +∞)
Estremo INCLUSO → parentesi quadra [ ]
Simboli: ≥ oppure ≤
Il pallino sulla retta è pieno ●
Es: x ≥ 3 → x ∈ [3, +∞)
⚠️ ±∞ ha sempre parentesi tonda — l'infinito non si raggiunge mai!
Es: x < 5 → (−∞, 5)  ·  x ≤ 5 → (−∞, 5]  ·  x > 2 → (2, +∞)  ·  x ≥ 2 → [2, +∞)
✏️ Esempio 1 — (3/4)x ≥ 16/9
(3/4)x ≥ 16/9
1
Riscrivo il secondo membro con la stessa base
16/9 = (4/3)² = (3/4)−2
← ribалto la frazione: 4/3 è l'inverso di 3/4, quindi (3/4)⁻²
↓
2
Equazione con stessa base (3/4)
(3/4)x ≥ (3/4)−2
↓
3
Applico la regola della base
La base è 3/4 → 0 < 3/4 < 1 → funzione DECRESCENTE
⚠️ Il verso SI INVERTE!

x ≥ −2  →  x ≤ −2
↓
4
Studio del segno — retta numerica
● pieno in −2 perché il simbolo è ≤ (estremo incluso)
🏆 SOLUZIONE
x ≤ −2  →  x ∈ (−∞, −2]
Parentesi quadra in −2 (incluso, simbolo ≤) · parentesi tonda in −∞ (infinito mai raggiunto)
✏️ Esempio 2 — (5/7)x > (5/7)1/2
(5/7)x > (5/7)1/2
1
Stessa base già pronta!
Base = 5/7 — già la stessa a sinistra e destra.
Posso passare direttamente agli esponenti.
↓
2
Applico la regola della base
La base è 5/7 → 0 < 5/7 < 1 → funzione DECRESCENTE
⚠️ Il verso SI INVERTE!

x > 1/2  →  x < 1/2
↓
3
Studio del segno — retta numerica
○ vuoto in 1/2 perché il simbolo è < (estremo escluso)
🏆 SOLUZIONE
x < 1/2  →  x ∈ (−∞, 1/2)
Parentesi tonde: 1/2 escluso (simbolo <) e −∞ mai raggiunto
✏️ Esempio 3 — 2x / (23x · 2⁴) − 8 ≤ 0
2x / (23x · 2⁴) − 8 ≤ 0
1
Semplificare il primo membro
2x23x · 2⁴ = 2x23x+4 = 2x−(3x+4) = 2x−3x−4 = 2−2x−4
← quoziente stessa base → sottraggo gli esponenti
↓
2
Riscrivo l'equazione
2−2x−4 − 8 ≤ 0
2−2x−4 ≤ 8
8 = 2³ → 2−2x−4 ≤ 2³
↓
3
Applico la regola della base
La base è 2 → 2 > 1 → funzione CRESCENTE
✓ Il verso NON cambia

−2x − 4 ≤ 3
−2x ≤ 7
Divido per −2 → il verso SI INVERTE (divisione per negativo!)
x ≥ −7/2
↓
4
Studio del segno — retta numerica
● pieno in −7/2 perché il simbolo è ≥ (estremo incluso)
🏆 SOLUZIONE
x ≥ −7/2  →  x ∈ [−7/2, +∞)
Parentesi quadra in −7/2 (incluso, simbolo ≥) · parentesi tonda in +∞
✏️ Esempio 4 — 7x ≥ √(7x−1)
7x ≥ √(7x−1)
1
Trasformo la radice in esponente frazionario
√(7x−1) = (7x−1)1/2 = 7(x−1)/2
← radice quadrata = esponente 1/2
↓
2
Stessa base — riscrivo
7x ≥ 7(x−1)/2
↓
3
Applico la regola della base
La base è 7 → 7 > 1 → funzione CRESCENTE
✓ Il verso NON cambia

x ≥ (x−1)/2
Moltiplico tutto per 2:   2x ≥ x − 1
2x − x ≥ −1
x ≥ −1
↓
4
Studio del segno — retta numerica
● pieno in −1 perché il simbolo è ≥ (estremo incluso)
🏆 SOLUZIONE
x ≥ −1  →  x ∈ [−1, +∞)
Parentesi quadra in −1 (incluso, simbolo ≥) · parentesi tonda in +∞
