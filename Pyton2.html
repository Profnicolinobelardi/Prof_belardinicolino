Python — Scheda introduttiva

Appunti di informatica - Prof. Belardi Nicolino

Python è un linguaggio di programmazione semplice da leggere e scrivere, nato nel 1991. Viene usato per analisi dati, siti web, intelligenza artificiale e molto altro. A differenza di C++, non serve dichiarare subito "che tipo" di dato usi: Python ci pensa da solo.

⚖️

Python vs C++ — le differenze principali

In C++ è obbligatorio indicare in anticipo il tipo di ogni variabile e rispettare una sintassi molto rigida. Python è più flessibile: si occupa da solo di capire che tipo di dato stai usando, e la sua scrittura è più vicina al linguaggio comune.

C++
// Variabile numero intero
int eta = 25;
// Variabile testo
string nome = "Marco";
cout << nome;
Python
# Variabile numero intero
eta = 25
# Variabile testo
nome = "Marco"
print(nome)
✔In Python non si scrive il tipo (int, string…): viene dedotto automaticamente.
✔Non servono i punto e virgola (;) alla fine di ogni riga.
✔Non servono le parentesi graffe {}: Python usa l'indentazione (gli spazi) per strutturare il codice.
✔Il codice gira riga per riga, senza bisogno di compilare prima.
📦

Dichiarazione di variabili

Una variabile è come una scatola con un'etichetta: dentro ci metti un dato, e la richiami usando il suo nome. In Python basta scegliere il nome e assegnarle un valore — nessuna altra formalità.

Due tipi fondamentali da conoscere: i numeri e le stringhe (testo).
🔢 Numerica 🔤 Stringa (testo)
Variabili numeriche — numeri interi e decimali
eta = 25 # numero intero (25 anni)
altezza = 1.75 # numero decimale (1,75 metri)
punteggio = 100 # numero intero (punteggio di gioco)
Variabili stringa — testo tra virgolette
nome = "Giulia" # testo tra virgolette doppie
citta = 'Roma' # va bene anche con virgolette singole
messaggio = "Ciao, mondo!" # testo con spazio e punteggiatura

La differenza visiva è semplice: i numeri si scrivono senza virgolette, il testo si scrive con virgolette. Se scrivi eta = "25", Python lo tratterà come testo, non come numero — e non potrai farci calcoli!

🔗

Assegnazione — cosa significa e come funziona

Assegnare significa collegare un nome (la variabile) a un valore. Lo fa il simbolo =. Attenzione: non è il simbolo di "uguale" in matematica — qui significa "metti questo valore in questa scatola".

Assegnazione numerica — esempi pratici
prezzo = 9.99 # nella scatola "prezzo" metto 9.99
quantita = 3 # nella scatola "quantita" metto 3
totale = prezzo * quantita # totale = 9.99 × 3 = 29.97
print(totale) # stampa il risultato
▶ 29.97
Assegnazione stringa — esempi pratici
nome = "Luca"
cognome = "Rossi"
nome_completo = nome + " " + cognome # unisce le stringhe
print(nome_completo)
▶ Luca Rossi
Nota: il valore di una variabile può cambiare nel tempo — puoi sovrascriverla quando vuoi. eta = 25 e poi più avanti eta = 26: ora "eta" vale 26.
🔀

La condizionale — prendere decisioni nel codice

Un condizionale dice al programma: "Se succede questa cosa, fai quest'altra cosa; altrimenti fai quest'altra ancora". È come un bivio su una strada. In Python si usa if / elif / else.

Esempio numerico — verifica l'età
eta = 17

if eta >= 18:
    print("Sei maggiorenne")
elif eta >= 14:
    print("Sei adolescente")
else:
    print("Sei bambino")
▶ Sei adolescente
Esempio stringa — saluto personalizzato
nome = "Admin"

if nome == "Admin":
    print("Benvenuto, amministratore!")
else:
    print("Ciao, " + nome)
▶ Benvenuto, amministratore!
Attenzione all'indentazione! Le righe dentro un if devono essere spostate a destra (di solito 4 spazi). È obbligatorio in Python: senza, il programma non capisce cosa fa parte della condizione.
📚

Le librerie — strumenti già pronti da usare

Una libreria è un insieme di funzionalità già scritte da altri che puoi "importare" nel tuo programma. È come prendere in prestito una cassetta degli attrezzi specializzata invece di costruire tutto da zero. Si usa la parola chiave import.

Esempio — usare la libreria math
import math # importo la libreria
risultato = math.sqrt(16) # calcola la radice quadrata
print(risultato)
▶ 4.0
