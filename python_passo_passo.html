Contare quanti numeri sono maggiori di 10

📄 Cosa dobbiamo fare?

Dobbiamo creare una scatola (lista) e metterci dentro dei numeri.
Poi dobbiamo controllare questi numeri uno ad uno e contare quanti di loro sono più grandi di 10.

🧠 Ragionamento Semplice (per l'alunno)

  • La scatola: Creo una lista vuota (una scatola vuota).
  • Quanti numeri: Chiedo quanti numeri vuoi inserire.
  • Inserimento: Con il ciclo for con range(0, n, 1) inserisco i numeri uno alla volta.
  • Controllo: Poi scorro la lista di nuovo con range(0, n, 1).
  • Confronto: Ogni volta controllo se il numero è maggiore di 10.
  • Il conto: Se sì, aumento il contatore.
  • Risultato: Alla fine stampiamo quanti numeri sono più grandi di 10.
💻 Il Codice Facile
🐍 Python
# Creo una lista vuota dove metterò i numeri
lista = []

# Chiedo all'alunno quanti numeri vuole inserire
n = int(input("Quanti numeri vuoi inserire? "))

# Riempio la lista usando il metodo classico con range
# range(0, n, 1) significa:
# start = 0 (parto da zero)
# stop = n (mi fermo prima di n)
# step = 1 (avanzo di uno alla volta)
for i in range(0, n, 1):
    numero = int(input("Inserisci un numero: "))
    lista.append(numero)

# Contatore dei numeri maggiori di 10
conta_maggiori = 0

# Scorro la lista usando di nuovo range e l'indice i
for i in range(0, n, 1):

    # Prendo l'elemento in posizione i
    x = lista[i]

    # Controllo se è maggiore di 10
    if x > 10:
        conta_maggiori = conta_maggiori + 1

# Stampo il risultato
print("Hai inserito", conta_maggiori, "numeri maggiori di 10")

📘 Spiegazione Didattica Riga per Riga

  • for i in range(0, n, 1):
    • start = 0 → parto dal primo elemento
    • stop = n → mi fermo prima di n
    • step = 1 → vado avanti di uno alla volta
  • lista.append(numero): Metto il numero dentro la lista.
  • x = lista[i]: Prendo l'elemento in posizione i.
  • if x > 10: Controllo se è più grande di 10.
  • conta_maggiori = conta_maggiori + 1: Aumento il contatore.

Contare i Numeri Pari e i Numeri Dispari

📄 Cosa dobbiamo fare?

Dobbiamo capire quanti numeri Pari e quanti numeri Dispari vengono inseriti.

I numeri Pari finiscono per: 0, 2, 4, 6, 8.
I numeri Dispari finiscono per: 1, 3, 5, 7, 9.

🧠 Ragionamento Passo-Passo

  • L'utente decide quanti numeri vuole scrivere (es. vuole inserire 4 numeri).
  • Chiediamo i numeri uno per volta usando un ciclo.
  • Per capire se un numero è Pari c'è una formula fissa che il computer capisce: x % 2 == 0.
  • Se la formula è vera, significa che il numero è Pari. E aggiungiamo 1 alla scatola dei Pari.
  • Se la formula è falsa, il numero è Dispari. E aggiungiamo 1 alla scatola dei Dispari.
  • Alla fine diciamo quanti Pari e Dispari abbiamo trovato.
💻 Il Codice Facile
🐍 Python
# Chiedo all'utente quanti numeri totali vuole scrivere
quantita = int(input("Quanti numeri vuoi inserire? "))

# Scatola per contare i numeri Pari (parte da 0)
pari = 0

# Scatola per contare i numeri Dispari (parte da 0)
dispari = 0

# Uso il ciclo for con range per ripetere l'azione:
# range(0, quantita, 1) significa:
# 0        -> punto di partenza (inizio)
# quantita -> fino a quando devo contare (fine)
# 1        -> il passo, cioè di quanto vado avanti ogni volta (+1)
for i in range(0, quantita, 1):

    # Chiedo il numero all'utente
    x = int(input("Inserisci un numero: "))

    # Formula fissa per controllare se è Pari
    if x % 2 == 0:
        # È pari! Aggiungo 1 alla scatola dei pari
        pari = pari + 1
    else:
        # È dispari! Aggiungo 1 alla scatola dei dispari
        dispari = dispari + 1

# Stampo i risultati finali
print("Numeri pari inseriti:", pari)
print("Numeri dispari inseriti:", dispari)

Il Gioco del 5 (Somma e Moltiplicazione)

📄 Cosa dobbiamo fare?

Ti daremo 6 numeri, uno dopo l'altro.

Per ogni numero, dobbiamo fare un controllo:
- Se il numero supera il 5 (per esempio: 6, 7, 8, 9...), lo aggiungiamo a una somma.
- Se il numero arriva fino a 5 (per esempio: 1, 2, 3, 4, 5), lo moltiplichiamo.

🧠 Ragionamento Passo-Passo

  • Iniziamo con una somma che vale 0 e una moltiplicazione (prodotto) che vale 1.
  • Chiediamo 6 numeri all'utente, usando un ciclo.
  • Quando l'utente inserisce un numero, guardiamo se è dal 6 in su usando il simbolo > 5.
  • Se è vero, facciamo una addizione (+).
  • Se invece è falso (cioè il numero è piccolo), facciamo una moltiplicazione (*).
  • Alla fine stampiamo i risultati.
💻 Il Codice Facile
🐍 Python
# La somma parte da 0 perché sommare inizia sempre da zero
somma = 0

# Il prodotto parte da 1 perché moltiplicare inizia sempre da uno
prodotto = 1

# Dobbiamo inserire 6 numeri in totale
contatore = 6

# Il ciclo si ripete finché il contatore non arriva a zero
while contatore > 0:

    # Chiedo all'utente di scrivere un numero
    x = float(input("Inserisci un numero: "))

    # Controllo: il numero è dal 6 in su? (> 5)
    if x > 5:
        # Se è dal 6 in su, lo aggiungo alla somma
        somma = somma + x
    else:
        # Altrimenti (se è 1, 2, 3, 4, 5), lo moltiplico
        prodotto = prodotto * x

    # Tolgo 1 al contatore, perché ho appena letto un numero
    contatore = contatore - 1

# Stampo i risultati finali
print("Somma dei numeri dal 6 in su:", somma)
print("Moltiplicazione dei numeri piccoli:", prodotto)

Contare i Voti (Promossi e Bocciati)

📄 Cosa dobbiamo fare?

Dobbiamo dividere i voti in due gruppi:

- Sufficienti: chi prende dal 6 in su (6, 7, 8, 9, 10). Significa che è Promosso.
- Insufficienti: chi prende dal 5 in giù (1, 2, 3, 4, 5). Significa che è Bocciato.

🧠 Ragionamento Passo-Passo

  • Prima chiediamo quanti voti dobbiamo controllare in totale.
  • Per ogni voto, controlliamo se è dal 6 in su usando >= 6.
  • Se è vero, significa che è Sufficiente. Aggiungiamo 1 ai Sufficienti.
  • Altrimenti, se è falso, significa che è Insufficiente. Aggiungiamo 1 agli Insufficienti.
  • Alla fine stampiamo quanti voti belli e quanti voti brutti ci sono stati.
💻 Il Codice Facile
🐍 Python
# Chiedo quanti voti in totale dobbiamo inserire
quantita = int(input("Quanti voti vuoi inserire? "))

# Scatola per contare i voti Sufficienti (Promossi)
sufficienti = 0

# Scatola per contare i voti Insufficienti (Bocciati)
insufficienti = 0

# Uso il ciclo for con range per ripetere l'azione:
# range(0, quantita, 1) significa:
# 0        -> punto di partenza (inizio)
# quantita -> fino a quando devo contare (fine)
# 1        -> il passo, cioè di quanto vado avanti ogni volta (+1)
for i in range(0, quantita, 1):

    # Chiedo di scrivere il voto
    voto = float(input("Inserisci un voto: "))

    # Controllo se il voto è dal 6 in su (>= 6)
    if voto >= 6:
        # Il voto è buono!
        sufficienti = sufficienti + 1
    else:
        # Il voto è brutto (dal 5 in giù)
        insufficienti = insufficienti + 1

# Stampo i risultati finali
print("Voti sufficienti (dal 6 in su):", sufficienti)
print("Voti insufficienti (dal 5 in giù):", insufficienti)

Creare una Lista di Numeri (La Scatola Magica)

📄 Cosa sono le Liste?

Le Liste in Python sono come un cassetto o uno scaffale dove puoi conservare tanti oggetti tutti insieme.

- Per creare una lista vuota usiamo le parentesi quadre: [ ]
- Quando vogliamo chiedere all'utente un numero, usiamo la parola input.
- Per infilare quel numero dentro la lista usiamo il comando append (che significa proprio "aggiungi in fondo").

🧠 Ragionamento Passo-Passo

  • Prima di tutto prepariamo una scatola vuota (la nostra lista).
  • Chiediamo per 3 volte all'utente di scrivere un numero, usando un ciclo.
  • Ogni volta che l'utente scrive un numero, lo prendiamo e lo buttiamo dentro la scatola usando .append().
  • Alla fine, apriamo la scatola e mostriamo (stampiamo) tutti i numeri che abbiamo raccolto.
💻 Il Codice Facile
🐍 Python
# 1. Preparo la scatola (lista) vuota usando le parentesi quadre
mia_lista = []

# 2. Ripeto l'azione per 3 volte usando range:
# range(0, 3, 1) significa: parto da 0, mi fermo prima di 3, vado avanti di 1
for i in range(0, 3, 1):

    # Chiedo un numero all'utente
    numero = int(input("Scrivi un numero da mettere nella scatola: "))

    # Metto il numero dentro la scatola usando "append"
    mia_lista.append(numero)

# 3. Stampo tutta la lista per vedere cosa c'è dentro
print("Ecco i numeri che hai inserito:", mia_lista)

Lavorare con 100 Numeri (Somma, Pari e Multipli)

📄 Cosa dobbiamo fare?

Creeremo una scatola (lista) e la riempiremo in automatico con i numeri da 1 a 100.

Poi dovremo fare tre cose su questi numeri:
1. Calcolare la somma di tutti i 100 numeri.
2. Contare quanti numeri NON sono multipli di 5 (cioè non fanno parte della tabellina del 5).
3. Raccogliere tutti i numeri Pari e metterli in una seconda scatola (una nuova lista).

🧠 Ragionamento Passo-Passo

  • Riempire la scatola: Invece di scrivere 100 numeri a mano, usiamo il ciclo for con range(1, 101, 1) che conta da solo da 1 fino a 100, e infiliamo ogni numero nella lista con .append().
  • La Somma: Partiamo da una somma vuota (0). Scorriamo la scatola e aggiungiamo ogni numero uno alla volta.
  • I multipli di 5: La formula fissa per la tabellina del 5 è x % 5 == 0. Visto che vogliamo quelli che NON ne fanno parte, usiamo il simbolo diverso: x % 5 != 0.
  • Seconda scatola: Prepariamo una scatola vuota per i numeri Pari. Scorriamo i numeri e, se un numero è Pari (x % 2 == 0), lo infiliamo nella nuova scatola con .append().
💻 Il Codice Facile
🐍 Python
# --- FASE 1: CREARE LA LISTA DA 1 A 100 ---

# Preparo una scatola (lista) vuota
lista1 = []

# Uso il ciclo for con range per ripetere l'azione:
# range(1, 101, 1) significa:
# 1   -> punto di partenza (inizio)
# 101 -> fino a quando devo contare (fine, si ferma a 100)
# 1   -> il passo, cioè di quanto vado avanti ogni volta (+1)
for numero in range(1, 101, 1):
    # Infilo il numero nella scatola usando append
    lista1.append(numero)

# Ora la "lista1" è piena!


# --- FASE 2: LA SOMMA ---

# La somma parte da 0
somma = 0

# Guardo ogni numero dentro la lista1
for x in lista1:
    # Aggiungo il numero alla somma totale
    somma = somma + x


# --- FASE 3: CONTARE I NUMERI CHE NON SONO NELLA TABELLINA DEL 5 ---

# Contatore che parte da 0
non_divisibili = 0

for x in lista1:
    # Se la formula del 5 NON (!=) è vera...
    if x % 5 != 0:
        # Aggiungo 1 al mio contatore
        non_divisibili = non_divisibili + 1


# --- FASE 4: CREARE LA SECONDA LISTA PER I PARI ---

# Preparo una seconda scatola (lista) vuota, solo per i Pari
lista_pari = []

for x in lista1:
    # Se il numero è Pari...
    if x % 2 == 0:
        # Lo infilo nella seconda scatola
        lista_pari.append(x)


# --- STAMPIAMO I RISULTATI FINALI ---

print("Somma di tutti i 100 numeri:", somma)
print("Quanti NON sono nella tabellina del 5:", non_divisibili)
print("Ecco la seconda scatola con i Pari:", lista_pari)

Unire le Liste e Lavorare con le Parole

📄 Cosa dobbiamo fare?

Questo esercizio ha due parti molto divertenti:

1. Incollare due liste: Abbiamo una lista con i numeri da 0 a 9 e un'altra con i numeri da 10 a 19. Dobbiamo incollarle insieme per farne una sola lunga.
2. Analizzare le parole: Creeremo una lista con tre parole ("sono", "una", "lista"). Chiederemo al computer di dirci di che tipo sono (cioè se sono testo) e quanto sono lunghe.

🧠 Ragionamento Passo-Passo

  • Incollare: In Python, per unire due liste si usa il simbolo del più (+). È come se mettessimo una lista dopo l'altra.
  • Controllo: Se incolliamo 0-9 e 10-19, otteniamo tutti i numeri da 0 a 19. Questo è esattamente quello che fa il comando range(20).
  • Parole e Lunghezza: Mettiamo le tre parole tra virgolette dentro le parentesi quadre. Poi, per ogni parola, usiamo il comando type() che serve a capire di che tipo è un oggetto (nel nostro caso è testo, cioè "stringa") e len() per contare quante lettere ha.
💻 Il Codice Facile
🐍 Python
# --- PARTE 1: INCOLLARE DUE LISTE ---

# range(10) significa creare una sequenza che parte da 0 e arriva a 9
lista_1 = list(range(10))

# range(10, 20) significa partire da 10 e arrivare fino a 19 (si ferma prima di 20)
lista_2 = list(range(10, 20))

# Le incollo insieme usando il simbolo +
lista_completa = lista_1 + lista_2

print("Lista unita:", lista_completa)
print("È uguale a range(20)? Sì, contiene i numeri da 0 a 19!")


# --- PARTE 2: LE PAROLE (STRINGHE) ---

# Creo una lista con tre parole
parole = ["sono", "una", "lista"]

# Uso un ciclo per guardare ogni parola
for p in parole:
    # type(p) dice al computer di che "famiglia" fa parte l'oggetto
    # len(p) conta quante lettere ci sono nella parola
    print("La parola '", p, "' è di tipo:", type(p), "ed è lunga:", len(p))

Contare quanti Pari e quanti Dispari

📄 Cosa dobbiamo fare?

Dobbiamo creare un programma che ci chiede quanti numeri vogliamo inserire.
Poi, per ogni numero che scriviamo, il computer deve capire se è Pari o Dispari e tenere il conto (come se avesse due barattoli per le caramelle).
Alla fine ci dirà quanti ne abbiamo messi in tutto.

🧠 Ragionamento Passo-Passo

  • La Quantità: Chiediamo all'inizio: "Quanti numeri vuoi controllare?".
  • I Contatori: Prepariamo due "barattoli" vuoti: uno chiamato pari e uno chiamato dispari. All'inizio valgono 0.
  • Il Ciclo: Usiamo un ciclo for per ripetere la domanda tante volte quante abbiamo deciso.
  • Il Controllo: Ogni volta controlliamo il numero con la formula x % 2 == 0.
    • Se è vero, aggiungiamo 1 al barattolo dei pari.
    • Altrimenti, aggiungiamo 1 al barattolo dei dispari.
  • Risultato: Alla fine stampiamo i numeri che abbiamo contato.
💻 Il Codice Facile
🐍 Python
# Chiedo quanti numeri vuoi inserire
# input() legge il testo, int() lo trasforma in un numero vero
quantita = int(input("Quanti numeri vuoi inserire? "))

# Preparo i due barattoli (contatori)
pari = 0
dispari = 0

# Inizia il ciclo che ripete la domanda tante volte:
# range(0, quantita, 1) significa:
# 0        -> punto di partenza (inizio)
# quantita -> fino a quando devo contare (fine)
# 1        -> il passo, cioè di quanto vado avanti ogni volta (+1)
for i in range(0, quantita, 1):
    
    # Chiedo un numero
    x = int(input("Inserisci un numero: "))
    
    # Controllo se è Pari usando il comando % 2
    # Se il resto della divisione per 2 è uguale a 0, il numero è Pari
    if x % 2 == 0:
        # Se è Pari, aumento il contatore
        pari = pari + 1
    else:
        # Se non è Pari, è Dispari!
        dispari = dispari + 1

# Stampo i risultati finali
print("Numeri pari inseriti:", pari)
print("Numeri dispari inseriti:", dispari)
