Esercizi — Equazioni esponenziali elementari
Per ogni esercizio: trova la base comune → riscrivi → uguaglia gli esponenti
6 3x = 1
3x = 1
Base di riferimento: 3
1 = 3⁰ → posso scrivere: 3x = 30
Stessa base → ugualio gli esponenti: x = 0
SOLUZIONE x = 0
7 5x = 25
5x = 25
Base di riferimento: 5
25 = 5² → 5x = 52
Stessa base → x = 2
SOLUZIONE x = 2
8 9x = 27
9x = 27
Base di riferimento: 3
9 = 3² → 9x = (3²)x = 32x
27 = 3³
Quindi: 32x = 33
Ugualio: 2x = 3 → x = 3/2
SOLUZIONE x = 3/2
9 8x = −3
8x = −3
Il numero a destra è negativo (−3).
Nessuna potenza può dare un numero negativo!
L'esponenziale è sempre positiva → impossibile
🚫 IMPOSSIBILE — nessun valore di x funziona
10 3x = 1/3
3x = 13
Base di riferimento: 3
13 = 3−1 (esponente negativo = vai sotto la frazione)
Quindi: 3x = 3−1
Ugualio: x = −1
SOLUZIONE x = −1
11 7x = 9
7x = 9
9 non è una potenza di 7 → non possiamo ricondurre alla stessa base con numeri interi.
In questo caso la soluzione si esprime con il logaritmo:
7x = 9 → x = log₇ 9
SOLUZIONE x = log₇ 9
12 2x = √2 / 64
2x = √264
Base di riferimento: 2
√2 = 21/2  ·  64 = 26
√264 = 21/226 = 21/2 − 6 = 2−11/2
Quindi: 2x = 2−11/2
Ugualio: x = −11/2
SOLUZIONE x = −11/2
13 8−x = 2

2 = 21
Quindi: 2−3x = 21
Ugualio: −3x = 1 → x = −1/3
SOLUZIONE x = −1/3

SOLUZIONE x = 0

SOLUZIONE x = −3
