SCHEDA 1 — Definizione e Condizioni di EsistenzaProf. Belardi Nicolino
Il logaritmo è l'operazione matematica che ci permette di trovare l'esponente a cui elevare una base per ottenere un determinato numero (chiamato argomento).
loga b = x  ⟺  ax = b
💡
A parole: il logaritmo in base a di b è l'esponente (x) che devo dare alla base (a) per ottenere l'argomento (b).
Condizioni di Esistenza (C.E.)
Base a
Deve essere positiva e diversa da 1

a > 1   oppure   0 < a < 1

Non può essere 0, 1 o negativa
Argomento b
Deve essere sempre positivo

b > 0

Non esiste log di 0 o di un numero negativo
Il Logaritmo Naturale (ln)
ln b = loge b

La scritta ln sta per logaritmo naturale (o logaritmo neperiano). Quando si usa ln, la base è il numero di Nepero (e).

Il numero di Nepero è una costante matematica irrazionale: e ≈ 2,71828…
È famoso come il Pi greco (π). Usare la base e semplifica enormemente formule e calcoli nello studio delle funzioni.
Il logaritmo come operazione inversa
Logaritmo, potenza e radice legano tre elementi: base, esponente, risultato.
A seconda dell'elemento incognito si usa un'operazione diversa.
Potenza
Cerco il Risultato
Conosco: base e esponente

2³ = 8
Radice
Cerco la Base
Conosco: esponente e risultato

∛8 = 2
Logaritmo
Cerco l'Esponente
Conosco: base e risultato

log₂ 8 = 3

SCHEDA 2 — Grafico e Proprietà AnaliticheProf. Belardi Nicolino
Caso 1 — Base maggiore di 1 (a > 1)
Grafico 1 — y = loga x con a > 1 · simmetrica a y = aˣ rispetto a y = x
●
Passaggio per il punto (1 ; 0)
Il grafico interseca l'asse x nel punto (1, 0).
Questo accade perché qualsiasi numero elevato a 0 dà 1 → a⁰ = 1.
Quindi l'esponente che bisogna dare ad a per ottenere 1 è sempre 0.
In formule: loga 1 = 0
●
Asintoto verticale
La curva scende verso il basso avvicinandosi all'asse y, ma senza mai toccarlo.
L'asse y (equazione x = 0) fa da "muro invisibile" → è l'asintoto verticale.
Il motivo è nelle C.E.: l'argomento del logaritmo deve essere sempre > 0.
La funzione non esiste per x = 0 né per x negativo → il grafico "vive" solo nella metà destra del piano.
●
Funzione inversa e simmetria
La funzione logaritmica e la funzione esponenziale sono l'una l'inversa dell'altra.

Il grafico di y = loga x e il grafico di y = aˣ sono simmetrici rispetto alla bisettrice del primo e terzo quadrante (retta y = x).
Caso 2 — Base compresa tra 0 e 1 (0 < a < 1)
Grafico 2 — y = loga x con 0 < a < 1 · funzione decrescente
●
Funzione decrescente
Quando 0 < a < 1, la funzione logaritmica è decrescente: all'aumentare di x, y diminuisce.
●
Simmetria anche in questo caso
Il grafico della funzione esponenziale decrescente (in verde) e quello della funzione logaritmica decrescente (in blu) sono perfettamente speculari rispetto alla retta bisettrice y = x.

SCHEDA 3 — Proprietà dei LogaritmiProf. Belardi Nicolino
⚠️ Tutte le proprietà sono valide solo se sono rispettate le C.E.: basi positive e diverse da 1, argomenti strettamente positivi.
Proprietà 1 — Logaritmo di un prodotto
loga b + loga c = loga (b · c)
La somma di due logaritmi con la stessa base è uguale a un unico logaritmo che ha per argomento il prodotto degli argomenti.
Proprietà 2 — Logaritmo di un quoziente
loga b − loga c = loga (b/c)
La differenza tra due logaritmi con la stessa base è uguale a un unico logaritmo che ha per argomento il quoziente degli argomenti.
Proprietà 3 — Logaritmo di una potenza
loga (bc) = c · loga b
Se l'argomento è elevato a un esponente, questo esponente può essere "portato fuori" e moltiplicato per il logaritmo. Utilissima per eliminare frazioni e radici dagli argomenti.
Esempio 1 (con frazioni):
log₂ (1/x) = log₂ (x−1) = −1 · log₂ x = −log₂ x

Esempio 2 (con radici):
log₂ ∛(x²) = log₂ (x2/3) = (2/3) · log₂ x
Proprietà 4 — Trasformare un numero in un logaritmo
n = loga (an)
Qualsiasi numero può essere riscritto come logaritmo scegliendo una base a piacere. La regola è: inserire come argomento la base scelta elevata a quel numero.
Esempio: il numero 5 scritto come logaritmo in base 2: 5 = log₂ (2⁵)
Con il logaritmo naturale: 5 = ln(e⁵)
Identità Fondamentale — Trasformare un numero in una potenza
Base 2
5 = 2log₂ 5
Base 3
5 = 3log₃ 5
Base e
5 = eln 5
✏️
SCHEDA 4 — Equazioni LogaritmicheProf. Belardi Nicolino
La strategia principale è usare le proprietà per arrivare alla forma: loga f(x) = loga g(x). Una volta ottenuto lo stesso logaritmo con la stessa base → si "tolgono" i logaritmi e si uguagliano gli argomenti: f(x) = g(x)
Equazioni Elementari — si usa direttamente la definizione
Esempio 1
log₂ x = 4
x = 2⁴ = 16
Esempio 2
log1/2 x = −1
x = (1/2)−1 = 2
Esempio 3
log₉ 81 = x
9x = 81 → x = 2
Equazioni da ricondurre alla forma base — si usano le proprietà
Esempio 4
log₂ (x+1) − log₂ 3 = 0
log₂ (x+1) = log₂ 3 → x + 1 = 3 → x = 2
Esempio 5
log₃ x + log₃ 5 = log₃ (x−4)
log₃ (5x) = log₃ (x−4) → 5x = x − 4 → x = −1
🚫 Verifica delle C.E. — Esempio 5: x = −1 renderebbe gli argomenti negativi → non accettabile. L'equazione è impossibile. Ricordare sempre di verificare le C.E. alla fine!
Esempi di utilizzo — trasformare i numeri
Esempio A
log₄ x = 32
log₄ x = log₄ (432) → x = 432
Esempio B — log₇ x = 0 (due metodi)
log₇ x = 0
1) 7⁰ = 1 → x = 1
2) log₇ x = log₇ (7⁰) → x = 1
🔑
Schema da seguire: usa le proprietà → loga f(x) = loga g(x) → uguaglia gli argomenti → verifica sempre le C.E.!
⚖️
SCHEDA 5 — Disequazioni LogaritmicheProf. Belardi Nicolino
Caso a > 1
Il verso si mantiene
f(x) > g(x)
Caso 0 < a < 1
Il verso si inverte
f(x) < g(x)
⚠️ C.E.: Argomenti sempre > 0.
✍️
