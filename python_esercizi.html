Calcola la media di tre voti (compresi tra 18 e 30) inseriti da tastiera. Se il voto non è valido, chiedilo di nuovo.

📄 Testo dell'esercizio

Calcola la media di tre voti inseriti da tastiera.
Ogni voto deve essere compreso tra 18 e 30, estremi inclusi (quindi 18 e 30 vanno bene).

Se l'utente inserisce un voto fuori da questo intervallo, il programma deve rifiutarlo e chiedere di nuovo, senza contarlo.

🧩 Ragionamento logico dell'esercizio

Obiettivo: Calcolare la media di tre voti inseriti da tastiera.
Ogni voto deve essere compreso tra 18 e 30, estremi inclusi.

Passi logici:
  • Devo chiedere tre voti all'utente.
  • Ogni volta che l'utente inserisce un voto:
    • se il voto è tra 18 e 30 → lo accetto e lo sommo
    • se il voto è fuori da questo intervallo → lo rifiuto e chiedo di nuovo (non lo conto)
  • Tengo: una somma dei voti validi e un contatore di quanti voti validi ho già inserito
  • Quando ho raccolto 3 voti validi, calcolo: media = somma / 3
  • Stampo la media finale.
🧩 Elementi principali di Python che useremo
1. Variabili
  • somma → serve per tenere la somma dei voti validi
  • cont → serve per contare quanti voti validi abbiamo già inserito
  • voto → è il singolo voto inserito dall'utente
  • media → è il risultato finale (la media)
2. Tipo int
  • int significa numero intero (senza virgola)
  • Lo usiamo per i voti perché i voti sono numeri interi (18, 19, 20, …, 30)
3. input()
  • input() serve per leggere da tastiera
  • Restituisce sempre una stringa (testo)
  • Per trasformarla in numero intero usiamo int(...)
Esempio:
Python
voto = int(input("Inserisci il voto: "))
  • mostra il messaggio
  • aspetta che l'utente scriva un numero
  • lo converte in intero e lo mette nella variabile voto
4. Ciclo while
  • while cont <= 3: significa: "ripeti il blocco finché cont è minore o uguale a 3"
  • Lo usiamo per assicurarci di raccogliere 3 voti validi
5. Operatori di confronto
  • 18 <= voto <= 30 significa: "il voto è maggiore o uguale a 18 e minore o uguale a 30"
6. Operatori di assegnazione
  • somma = 0 → inizializzo la somma a 0
  • somma += voto → è uguale a somma = somma + voto
  • cont = 1 → parto dal primo voto
  • cont += 1 → è uguale a cont = cont + 1
💻 Codice riscritto e commentato riga per riga
Python
# Calcoliamo il valore medio di tre voti inseriti da tastiera.
# Ogni voto deve essere compreso tra 18 e 30 (inclusi).

somma = 0
# Inizializzo la variabile somma a 0.
# somma servirà per accumulare (somma = somma + voto) tutti i voti validi.
# Parto da 0 perché all'inizio non ho ancora aggiunto nessun voto.

cont = 1
# Inizializzo il contatore a 1.
# cont rappresenta "il numero del voto" che sto chiedendo (primo, secondo, terzo).
# Parto da 1 perché sto per chiedere il primo voto.

while cont <= 3:
    # Il ciclo while ripete il blocco di istruzioni finché la condizione è vera.
    # Qui la condizione è: cont <= 3
    # Significa: continua a chiedere voti finché non hai inserito 3 voti validi.
    # Appena cont diventa 4, la condizione diventa falsa e il ciclo si ferma.

    voto = int(input("Inserisci il voto: "))
    # input("Inserisci il voto: ") mostra il messaggio e aspetta che l'utente scriva qualcosa.
    # input restituisce una stringa (testo), ad esempio "25".
    # int(...) converte quella stringa in un numero intero, ad esempio 25.
    # Il valore intero viene memorizzato nella variabile voto.

    if 18 <= voto <= 30:
        # Questa è una struttura condizionale if.
        # La condizione 18 <= voto <= 30 significa:
        # "il voto è maggiore o uguale a 18 E minore o uguale a 30".
        # Se il voto è dentro l'intervallo [18, 30], allora il blocco sotto viene eseguito.

        somma += voto
        # somma += voto è una forma abbreviata di: somma = somma + voto.
        # Aggiungo il voto corrente alla somma totale dei voti validi.
        # In questo modo, somma conterrà sempre la somma di tutti i voti validi inseriti finora.

        cont += 1
        # cont += 1 è una forma abbreviata di: cont = cont + 1.
        # Aumento il contatore di 1 perché ho appena accettato un voto valido.
        # Quando cont arriverà a 4, il while si fermerà (perché cont <= 3 sarà falso).

    else:
        # Questo blocco viene eseguito se la condizione dell'if è falsa,
        # cioè se il voto NON è compreso tra 18 e 30.

        print("Voto non valido")
        # Avviso l'utente che il voto inserito non è valido.
        # In questo caso NON aumento cont e NON aggiungo il voto alla somma.
        # Quindi il ciclo while chiederà di nuovo un voto per sostituire quello non valido.

# Quando il while finisce, significa che abbiamo raccolto 3 voti validi.
# A questo punto possiamo calcolare la media.

media = somma / 3
# Calcolo la media dividendo la somma totale dei voti per 3.
# Uso 3 perché l'esercizio richiede la media di tre voti.
# Il risultato può essere anche un numero con la virgola (float).

print("La media finale è:", media)
# Stampo a schermo la frase "La media finale è:" seguita dal valore della media.
# L'alunno vede il risultato del calcolo.

Il titolare di una cartoleria vuole calcolare l'incasso totale della giornata e il prezzo medio degli articoli venduti.

📄 Testo dell'esercizio

Il titolare di una cartoleria vuole un programma che calcoli:
  • l'incasso totale della giornata
  • il prezzo medio degli articoli venduti

L'utente deve dire quanti articoli ha venduto. Poi deve inserire uno alla volta i prezzi di ogni articolo.

🧠 Ragionamento logico dell'esercizio

Obiettivo: calcolare l'incasso totale e il prezzo medio degli articoli venduti in una cartoleria.

✔ Passi logici:
  • Chiedo all'utente quanti articoli ha venduto (es. 5 articoli).
  • Devo leggere un prezzo alla volta.
  • Ogni volta che l'utente inserisce un prezzo:
    • lo aggiungo alla somma totale
    • diminuisco il numero degli articoli rimanenti da leggere
  • Quando ho letto tutti i prezzi:
    • calcolo l'incasso totale (è la somma)
    • calcolo la media: media = somma / n
  • Stampo: l'incasso totale e il prezzo medio.
🧩 Elementi principali di Python che useremo
1. Variabili
  • somma → serve per tenere la somma di tutti i prezzi
  • media → sarà il risultato finale
  • articoli → quanti articoli devo ancora leggere
  • n → serve per ricordare il numero iniziale di articoli
  • prezzo → è il singolo prezzo inserito dall'utente
2. Tipo int e tipo float
  • int → numero intero (es. 5, 10, 20). Lo usiamo per il numero di articoli.
  • float → numero con la virgola (es. 2.50, 1.99, 10.00). Lo usiamo per i prezzi, perché i prezzi possono avere i centesimi.
3. input()
  • input() legge da tastiera e restituisce sempre testo.
Esempio:
Python
articoli = int(input("inserire numero articoli: "))
  • mostra il messaggio
  • aspetta che l'utente scriva qualcosa
  • int(...) trasforma il testo in numero intero
  • assegna il numero alla variabile articoli
4. Ciclo while
Il ciclo:
Python
while articoli > 0:
  • significa: "continua a chiedere prezzi finché ci sono articoli da leggere"
  • ogni volta che leggo un prezzo valido: lo aggiungo alla somma e diminuisco articoli di 1
  • quando articoli diventa 0 → il ciclo finisce
5. Operatori di assegnazione
  • somma = 0 → inizializzo la somma
  • somma += prezzo → aggiungo il prezzo alla somma
  • articoli -= 1 → diminuisco il numero di articoli rimanenti
  • media = somma / n → calcolo la media
💻 Codice riscritto e commentato riga per riga
Python
# Il titolare di una cartoleria vuole calcolare:
# 1) l'incasso totale della giornata
# 2) il prezzo medio degli articoli venduti

media = 0.00
# Inizializzo la variabile media a 0.00.
# Uso 0.00 perché la media sarà un numero con la virgola (float).
# All'inizio non ho ancora calcolato nulla, quindi parto da zero.

somma = 0
# Inizializzo la variabile somma a 0.
# somma servirà per accumulare tutti i prezzi degli articoli.
# Parto da 0 perché all'inizio non ho ancora aggiunto nessun prezzo.

articoli = int(input("inserire numero articoli: "))
# input("inserire numero articoli: ") mostra il messaggio e aspetta che l'utente scriva qualcosa.
# input restituisce una stringa (testo), ad esempio "5".
# int(...) converte quella stringa in un numero intero, ad esempio 5.
# Il numero viene memorizzato nella variabile articoli.
# articoli rappresenta quanti prezzi devo ancora leggere.

n = articoli
# Salvo il numero iniziale degli articoli nella variabile n.
# Questo è importante perché articoli verrà modificata nel ciclo (articoli -= 1).
# n mi serve per calcolare la media alla fine (media = somma / n).

while articoli > 0:
    # Il ciclo while ripete il blocco finché la condizione è vera.
    # Qui la condizione è: articoli > 0
    # Significa: continua a chiedere prezzi finché ci sono articoli da leggere.
    # Ogni volta che leggo un prezzo valido, articoli diminuirà di 1.

    prezzo = float(input("inserire prezzo articolo: "))
    # input("inserire prezzo articolo: ") chiede all'utente di inserire il prezzo.
    # input restituisce una stringa, ad esempio "2.50".
    # float(...) converte la stringa in un numero con la virgola (float), ad esempio 2.50.
    # Il valore viene memorizzato nella variabile prezzo.

    somma += prezzo
    # somma += prezzo è una forma abbreviata di: somma = somma + prezzo.
    # Aggiungo il prezzo dell'articolo alla somma totale.
    # In questo modo somma conterrà sempre l'incasso totale aggiornato.

    articoli -= 1
    # articoli -= 1 è una forma abbreviata di: articoli = articoli - 1.
    # Diminuisco il numero di articoli rimanenti da leggere.
    # Quando articoli diventa 0, il ciclo while si fermerà.

media = somma / n
# Calcolo la media dividendo la somma totale dei prezzi per n.
# Uso n perché rappresenta il numero iniziale degli articoli venduti.
# Il risultato sarà un numero con la virgola (float).

print("incasso totale: ", somma)
# Stampo l'incasso totale della giornata, cioè la somma di tutti i prezzi.

print("media: ", media)
# Stampo la media dei prezzi degli articoli venduti.

Il giornalaio vuole trovare il prezzo massimo tra tutti i prezzi degli articoli venduti nella giornata.

📄 Testo dell'esercizio

Il giornalaio vuole un programma che:
  • chiede quanti articoli ha venduto
  • chiede il prezzo di ogni articolo
  • trova il prezzo massimo, cioè il più alto tra quelli inseriti

🧠 Ragionamento logico dell'esercizio

Obiettivo: trovare il prezzo massimo tra tutti i prezzi degli articoli venduti.

✔ Passi logici:
  • Chiedo all'utente quanti articoli vuole inserire.
  • Devo leggere un prezzo alla volta.
  • Ogni volta che l'utente inserisce un prezzo:
    • lo confronto con il prezzo massimo trovato finora
    • se è più grande → diventa il nuovo prezzo massimo
  • Ripeto finché ho letto tutti gli articoli.
  • Alla fine stampo il prezzo massimo.
🧩 Elementi principali di Python che useremo
1. Variabili
  • prezzo → contiene il prezzo dell'articolo corrente
  • prezzomassimo → contiene il prezzo più alto trovato finora
  • cont → conta quanti articoli ho già inserito
  • articoli → quanti articoli devo leggere in totale
2. Tipo int
  • Usiamo int perché il numero degli articoli è un numero intero
  • Il prezzo, in questo esercizio, è trattato come intero (es. 2, 3, 5)
3. input()
  • input() legge da tastiera e restituisce testo. Per trasformarlo in numero intero usiamo int(input(...))
Esempio:
Python
prezzo = int(input("inserisci il prezzo "))
  • mostra il messaggio
  • aspetta che l'utente scriva un valore
  • int() trasforma il testo in numero
  • salva il numero nella variabile prezzo
4. Ciclo while
Il ciclo:
Python
while cont < articoli:
  • significa: "continua finché non hai letto tutti gli articoli"
  • ogni giro: leggo un prezzo, confronto con il massimo, aumento il contatore
5. Operatori di confronto
Python
if prezzo > prezzomassimo:
  • significa: "se il prezzo è più grande del massimo attuale…"
6. Operatori di assegnazione
  • prezzomassimo = 0 → inizializzo il massimo
  • prezzomassimo = prezzo → aggiorno il massimo
  • cont = cont + 1 → aumento il contatore
💻 Codice riscritto e commentato riga per riga
Python
# Il giornalaio vuole un programma che trovi il prezzo massimo
# tra tutti i prezzi degli articoli venduti nella giornata.

prezzo = 0
# Inizializzo la variabile prezzo a 0.
# prezzo conterrà il valore dell'articolo corrente inserito dall'utente.
# Parto da 0 perché ancora non ho letto nessun prezzo.

prezzomassimo = 0
# Inizializzo il prezzo massimo a 0.
# prezzomassimo servirà per memorizzare il prezzo più alto trovato finora.
# Parto da 0 perché non ho ancora confrontato nessun prezzo.
# Ogni volta che trovo un prezzo più alto, aggiornerò questa variabile.

cont = 0
# Inizializzo il contatore a 0.
# cont rappresenta quanti articoli ho già inserito.
# Parto da 0 perché non ho ancora letto nessun articolo.

articoli = int(input("inserisci n articoli "))
# input("inserisci n articoli ") mostra il messaggio e aspetta che l'utente scriva qualcosa.
# input restituisce una stringa, ad esempio "5".
# int(...) converte la stringa in un numero intero, ad esempio 5.
# Il numero viene memorizzato nella variabile articoli.
# articoli rappresenta quanti prezzi devo leggere in totale.

while cont < articoli:
    # Il ciclo while ripete il blocco finché la condizione è vera.
    # Qui la condizione è: cont < articoli
    # Significa: continua a chiedere prezzi finché non hai letto tutti gli articoli.
    # Quando cont diventa uguale ad articoli, il ciclo si ferma.

    prezzo = int(input("inserisci il prezzo "))
    # input("inserisci il prezzo ") chiede all'utente di inserire il prezzo dell'articolo.
    # input restituisce una stringa, ad esempio "3".
    # int(...) converte la stringa in un numero intero, ad esempio 3.
    # Il valore viene memorizzato nella variabile prezzo.

    if prezzo > prezzomassimo:
        # Questa è una struttura condizionale if.
        # La condizione prezzo > prezzomassimo significa:
        # "il prezzo inserito è maggiore del prezzo massimo trovato finora?"
        # Se la condizione è vera, allora aggiorno il prezzo massimo.

        prezzomassimo = prezzo
        # Aggiorno il prezzo massimo mettendo dentro prezzomassimo il valore di prezzo.
        # In questo modo prezzomassimo conterrà sempre il valore più alto inserito finora.

    cont = cont + 1
    # Aumento il contatore di 1.
    # Significa che ho letto un articolo in più.
    # Quando cont raggiungerà il valore di articoli, il ciclo while si fermerà.

print("il prezzo massimo è", prezzomassimo)
# Stampo il prezzo massimo trovato.
# L'utente vedrà il valore più alto tra tutti quelli inseriti.
