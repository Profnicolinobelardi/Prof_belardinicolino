Esercizi svolti passo per passo
⚠️ Ricorda sempre questa regola prima di iniziare
📈 Base MAGGIORE di 1
Es: 2, 3, 5/2, 7…
Il verso NON cambia
af(x) > ag(x)
→ f(x) > g(x)
📉 Base tra 0 e 1
Es: 1/2, 2/3, 3/4…
Il verso SI INVERTE !
af(x) > ag(x)
→ f(x) < g(x)
📏 Come scrivere la soluzione — parentesi tonde o quadre?
[ ] Parentesi QUADRA
Si usa con ≥ oppure ≤
→ il numero è COMPRESO
→ pallino ● pieno sulla retta

Es: x ≥ 2 → [2, +∞)
Es: x ≤ 2 → (−∞, 2]
( ) Parentesi TONDA
Si usa con > oppure <
→ il numero NON è compreso
→ pallino ○ vuoto sulla retta

Es: x > 2 → (2, +∞)
Es: x < 2 → (−∞, 2)
💡
∞ ha sempre la parentesi tonda — l'infinito non si raggiunge mai, quindi non può mai essere "compreso".

!
Quando vedi il numero 1 a destra…
Qualsiasi numero elevato a 0 dà sempre 1.
Quindi posso sempre scrivere il numero 1 come b⁰, usando la stessa base che ho a sinistra.

In questo caso:
10x = 1 = 10⁰

Stessa base → ugualio gli esponenti:
x = 0
