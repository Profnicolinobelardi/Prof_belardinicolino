ESERCIZIO 1 – Equazioni esponenziali

Formula Generale e Regola

Un'equazione esponenziale si presenta nella forma: \(a^{f(x)} = a^{g(x)}\)
La regola d'oro è: se le basi sono uguali, possiamo uguagliare gli esponenti.
Quindi la formula risolutiva è: \(a^{f(x)} = a^{g(x)} \implies f(x) = g(x)\)
(L'obiettivo è sempre fare in modo di avere la stessa base a destra e a sinistra dell'uguale).
Esempi Passo Passo:
a
Esempio: \(2^x = 16\)
Trasforma nella stessa base: Sappiamo che 16 è uguale a \(2^4\).
Riscrivi: \(2^x = 2^4\)
Uguaglia gli esponenti: \(x = 4\)
b
Esempio: \(3^{x+1} = 27\)
Trasforma nella stessa base: Sappiamo che 27 è \(3^3\).
Riscrivi: \(3^{x+1} = 3^3\)
Uguaglia gli esponenti: \(x + 1 = 3\)
Risolvi: \(x = 3 - 1 \implies x = 2\)
c
Esempio: \(5^{2x} = 1\)
Trasforma nella stessa base: Ricorda che qualsiasi numero elevato a zero fa 1 (\(a^0 = 1\)). Quindi \(1 = 5^0\).
Riscrivi: \(5^{2x} = 5^0\)
Uguaglia gli esponenti: \(2x = 0 \implies x = 0\)
d
Esempio: \(4^x = 8\)
Trasforma nella stessa base: Qui dobbiamo trasformare entrambe le basi in base 2. Sappiamo che \(4 = 2^2\) e \(8 = 2^3\).
Riscrivi: \((2^2)^x = 2^3 \implies 2^{2x} = 2^3\)
Uguaglia gli esponenti: \(2x = 3 \implies x = \frac{3}{2}\)
Per ogni equazione, trova la soluzione:
a
Esercizio: \(4^x = \dfrac{1}{4}\)
Trasforma nella stessa base: Ricorda che \(\frac{1}{4} = 4^{-1}\) (esponente negativo = reciproco).
Riscrivi: \(4^x = 4^{-1}\)
Uguaglia gli esponenti: \(x = -1\)
b
Esercizio: \(2^x = 16\)
Trasforma nella stessa base: Sappiamo che \(16 = 2^4\).
Riscrivi: \(2^x = 2^4\)
Uguaglia gli esponenti: \(x = 4\)
c
Esercizio: \(9^x = 27\)
Trasforma nella stessa base: Entrambe le basi si riducono in base 3. Sappiamo che \(9 = 3^2\) e \(27 = 3^3\).
Riscrivi: \((3^2)^x = 3^3 \implies 3^{2x} = 3^3\)
Uguaglia gli esponenti: \(2x = 3 \implies x = \dfrac{3}{2}\)
d
Esercizio: \(3^x = \sqrt[3]{3^2}\)
Trasforma la radice in esponente frazionario: Usando la regola \(\sqrt[n]{a^m} = a^{m/n}\), con \(n=3\) e \(m=2\):
\(\sqrt[3]{3^2} = 3^{2/3}\)
Riscrivi: \(3^x = 3^{2/3}\)
Uguaglia gli esponenti: \(x = \dfrac{2}{3}\)
ESEMPIO 1
\(\sqrt[3]{a^2} = a^{2/3}\)
indice=3, esponente=2 → 2/3

ESERCIZIO 2 – Disequazioni esponenziali

Regola Generale

Tutto dipende dalla base \(a\):
  • Se \(a > 1\): Il verso della disequazione NON CAMBIA.
    \(a^{f(x)} > a^{g(x)} \implies f(x) > g(x)\)
  • Se \(0 < a < 1\) (es. base 1/2): Il verso della disequazione CAMBIA.
    \(a^{f(x)} > a^{g(x)} \implies f(x) < g(x)\)
Esempi Passo Passo:
a
Esempio con Base > 1: \(2^x > 8\)
Risoluzione: \(2^x > 2^3\). Poiché la base è 2 (maggiore di 1), il verso non cambia: \(x > 3\).
Tabella dei segni (o grafico lineare): Tiriamo una linea per i numeri. A destra di 3 mettiamo il segno + (dove la condizione è vera), a sinistra -.
------- (3) +++++++
Intervallo: \((3, +\infty)\)
b
Esempio con Base compresa tra 0 e 1: \((1/2)^x \ge 1/4\)
Risoluzione: Sappiamo che \(1/4 = (1/2)^2\). Quindi: \((1/2)^x \ge (1/2)^2\).
Poiché la base è 1/2 (minore di 1), il verso cambia: \(x \le 2\).
Tabella dei segni:
+++++++ (2) -------
(la soluzione è la parte positiva, a sinistra di 2).
Intervallo: \((-\infty, 2]\) (usiamo la parentesi quadra perché c'è il simbolo di "uguale" \(\ge\)).

ESERCIZIO 3 – Funzioni esponenziali e grafico

Regole Generali

La funzione esponenziale è \(y = a^x\)
  • Passa SEMPRE per il punto \((0, 1)\), perché \(a^0 = 1\).
  • L'asintoto orizzontale è l'asse delle \(x\) (retta \(y = 0\)). Significa che il grafico si avvicina sempre di più all'asse \(x\) senza mai toccarlo.
Esempi:
a) \(y = 2^x\) (Base > 1)

La curva parte da molto vicino all'asse \(x\) (a sinistra), incrocia l'asse \(y\) in \((0, 1)\) e poi sale rapidissimamente verso l'alto (crescente).
b) \(y = (1/2)^x\) (Base < 1)

La curva scende da molto in alto (a sinistra), incrocia l'asse \(y\) sempre nel punto \((0, 1)\) e si appiattisce verso destra avvicinandosi all'asse \(x\) (decrescente).
🪵
Funzioni logaritmiche

Regole Generali

La funzione logaritmica \(y = \log_a(x)\) è l'inversa dell'esponenziale.
  • Passa SEMPRE per il punto \((1, 0)\), perché \(\log_a(1) = 0\).
  • L'asintoto verticale è l'asse delle \(y\) (cioè \(x = 0\)).
  • Condizione di esistenza (Dominio): \(x > 0\).
I tuoi casi:
a) \(y = \log_2(x)\)

Poiché la base \(2 > 1\), la curva sale dal basso (vicino all'asse \(y\)), passa per \((1, 0)\) e continua a crescere lentamente verso destra.
b) \(y = \log_{1/2}(x)\)

Poiché la base \(1/2 < 1\), la curva scende dall'alto (vicino all'asse \(y\)), passa per \((1, 0)\) e decresce lentamente verso destra.
📐
Proprietà dei Logaritmi
⚠️ Tutte le proprietà sono valide solo se sono rispettate le C.E.: basi positive e diverse da 1, argomenti strettamente positivi.
Proprietà 1 — Logaritmo di un prodotto
loga b + loga c = loga (b · c)
La somma di due logaritmi con la stessa base è uguale a un unico logaritmo che ha per argomento il prodotto degli argomenti.
Esempio: log₂ 4 + log₂ 8 = log₂ (4 · 8) = log₂ 32 = 5
Proprietà 2 — Logaritmo di un quoziente
loga b − loga c = loga (b/c)
La differenza tra due logaritmi con la stessa base è uguale a un unico logaritmo che ha per argomento il quoziente degli argomenti.
Esempio: log₃ 27 − log₃ 9 = log₃ (27 / 9) = log₃ 3 = 1
Proprietà 3 — Logaritmo di una potenza
loga (bc) = c · loga b
Se l'argomento è elevato a un esponente, questo esponente può essere "portato fuori" e moltiplicato per il logaritmo. Utilissima per eliminare frazioni e radici dagli argomenti.
Esempio 1 (con frazioni):
log₂ (1/x) = log₂ (x−1) = −1 · log₂ x = −log₂ x

Esempio 2 (con radici):
log₂ ∛(x²) = log₂ (x2/3) = (2/3) · log₂ x
Proprietà 4 — Trasformare un numero in un logaritmo
n = loga (an)
Qualsiasi numero può essere riscritto come logaritmo scegliendo una base a piacere. La regola è: inserire come argomento la base scelta elevata a quel numero.
Esempio: il numero 5 scritto come logaritmo in base 2: 5 = log₂ (2⁵)
Con il logaritmo naturale: 5 = ln(e⁵)
Identità Fondamentale — Trasformare un numero in una potenza
Base 2
5 = 2log₂ 5
Base 3
5 = 3log₃ 5
Base e
5 = eln 5
✏️
Equazione logaritmica
Testo: \(\log_2(2x - 3) = \log_2 2\)
1
Passo 1: Controllo base e forma
Sì, i logaritmi hanno la stessa base (2) e sono "elementari" (uno a destra e uno a sinistra dell'uguale). La regola dice che se \(\log_a f(x) = \log_a g(x)\), allora posso uguagliare gli argomenti: \(f(x) = g(x)\)
2
Passo 2: Condizione di Esistenza (C.E.)
L'argomento del logaritmo deve sempre essere maggiore di zero.
\(2x - 3 > 0\)
\(2x > 3 \implies x > \frac{3}{2}\)
(Questa condizione significa che la soluzione finale, per essere accettabile, dovrà essere più grande di 1,5).
3
Passo 3: Calcolo dell'equazione
Uguagliamo gli argomenti:
\(2x - 3 = 2\)
Spostiamo il -3 a destra cambiandolo di segno:
\(2x = 2 + 3\)
\(2x = 5\)
Dividiamo per 2: \(x = \frac{5}{2}\)
4
Passo 4: Verifica con la C.E.
La nostra soluzione è \(5/2\) (cioè 2,5). La C.E. ci chiedeva \(x > 3/2\) (cioè \(x > 1,5\)).
Poiché 2,5 > 1,5, la soluzione è accettabile.
