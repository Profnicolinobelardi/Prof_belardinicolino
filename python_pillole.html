🟦 1. Cosa sono le variabili in Python
Python è un linguaggio di programmazione moderno e semplice da leggere.

A differenza di C++, in Python:
  • ❌ Non usiamo le parentesi graffe { } per i blocchi di codice — al loro posto usiamo l'indentazione (rientri obbligatori)
  • ❌ Non dobbiamo dichiarare il tipo della variabile (niente int a;)
  • ✅ Dopo ogni condizione o ciclo mettiamo i due punti :
  • ✅ Python capisce da solo che tipo è il valore che assegniamo
  • ✅ Il codice è molto più leggibile e vicino all'italiano/inglese
In Python non serve dichiarare il tipo della variabile (come in C++). La variabile nasce nel momento in cui le assegni un valore.
Python
a = 12
b = "casa"
  • a contiene un numero → Python capisce da solo che è un intero
  • b contiene testo → Python capisce da solo che è una stringa
👉 Python capisce il tipo da solo. Tu devi solo scrivere il valore.
Attenzione all'input:

Quando scriviamo:
Python
x = int(input("inserisci un numero --> "))
Non stiamo dichiarando la variabile: stiamo dicendo a Python di leggere qualcosa da tastiera (input()), e poi di convertirla in un numero intero (int()) prima di salvarla in x.

In pratica: "prendi quello che digita l'utente e trattalo come un numero intero".
🟦 2. La stampa a schermo: print()
Per far vedere qualcosa sullo schermo si usa la funzione print():
Python
print("ciao")
Puoi stampare numeri, parole, variabili:
Python
a = 12
print(a)
print("il valore di a è", a)
👉 Dentro print() puoi mettere più cose separate da virgole e le stamperà tutte di fila con uno spazio.
🟦 3. Indentazione: cosa significa?
⚠ Questo è IMPORTANTISSIMO in Python.
Indentazione significa "spostare verso destra" una riga di codice usando gli spazi (di solito 4 spazi o un TAB).
Python
if a > 10:
    print("ciao")
La riga print("ciao") è indentata (spostata a destra).

In Python l'indentazione non è estetica, è obbligatoria. Serve per dire quali istruzioni appartengono a un blocco (if, while, for…).

Se togli l'indentazione, il programma non funziona e dà errore.
👉 Dove in C++ usavi le { }, in Python usi i rientri (spazi). È il modo di Python per "raggruppare" le istruzioni.
🟦 4. La struttura condizionale: IF
Serve per fare una scelta. Se una condizione è vera, esegue un blocco di istruzioni.
Python
a = 12

if a > 10:
    print(a)

print("ciao")
  • Se a > 10 è vera → stampa a
  • Poi stampa sempre "ciao" perché è fuori dall'if (non è indentata)
👉 L'IF finisce quando finisce l'indentazione. Tutto ciò che non è rientrato è "fuori" dall'if.
🟦 5. Il ciclo WHILE
Il ciclo while ripete le istruzioni finché la condizione è vera.
Python
a = 12

while a > 10:
    print(a)
    a -= 1   # diminuisce di 1 ogni giro

print("fine")
Cosa succede passo per passo:
  • stampa 12
  • stampa 11
  • poi a diventa 10 → la condizione a > 10 diventa falsa
  • il ciclo si ferma
  • stampa "fine"
👉 Anche qui l'indentazione indica cosa fa parte del ciclo. print("fine") non è indentato → è fuori dal while.
🟦 6. Il ciclo FOR e la funzione range()
Il ciclo for ripete un numero preciso di volte.

Dentro le parentesi di range() mettiamo fino a tre valori: start (partenza), stop (fine, esclusa) e step (il passo, cioè di quanto aumenta ogni volta).
Se si parte da 0 e il passo è 1, si può scrivere in forma semplice con solo il numero finale.
Python
for i in range(10):
    print("ciao")
Questo stampa "ciao" 10 volte.
🔍 Come funziona range()?
✔ range(10)
Parte da 0 e arriva a 9
(10 numeri totali)
✔ range(5, 12)
Parte da 5 e arriva a 11
(12 è escluso)
✔ range(0, 101, 1)
Parte da 0, arriva a 100, aumenta di 1 ogni volta
🟦 7. Esempio: sommare i primi 100 numeri
Python
somma = 0

for i in range(0, 101, 1):
    somma += i

print(somma)
Il ciclo percorre tutti i numeri da 0 a 100 (101 è escluso) e li aggiunge uno alla volta alla variabile somma.
Alla fine print(somma) mostrerà 5050.
🟦 8. Esempio completo: somma dei negativi e prodotto dei non negativi
Python
somma = 0
prodotto = 1

for i in range(5):
    x = int(input("inserisci un numero --> "))

    if x >= 0:
        prodotto = prodotto * x
    else:
        somma = somma + x

print("la somma dei valori negativi è", somma)
print("il prodotto dei valori non negativi è", prodotto)
Spiegazione semplice:
  • Il programma chiede 5 numeri all'utente (il for gira 5 volte)
  • Se il numero è positivo o zero → lo moltiplica nel prodotto
  • Se il numero è negativo → lo aggiunge alla somma
  • Alla fine stampa i due risultati
