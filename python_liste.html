Le Liste in
Python

Scheda Didattica Prof. Belardi Nicolino
Le liste in Python sono come degli scaffali o dei cassetti organizzati, dove puoi riporre più oggetti (i dati) usando un solo nome. Immagina di avere una scatola grande con tanti scompartimenti: invece di creare una variabile diversa per ogni informazione, puoi raccogliere tutto all'interno di una singola lista. Le caratteristiche principali sono la flessibilità, la capacità di contenere tipi di dato diversi e la possibilità di crescere o rimpicciolirsi durante l'esecuzione del programma. Ora le andremo a vedere nello specifico.
0 Riepilogo Rapido: Sintassi Essenziale

Creare una lista

Una lista si crea con le parentesi quadre [ ]. Può contenere dati di tipo diverso nella stessa struttura.

numeri = [10, 20, 30, 40, 50] mista = ["Alice", 3.14, True, 42] vuota = []

Accedere agli elementi

Gli indici partono da 0. Gli indici negativi partono dalla fine della lista.

numeri[0] # -> 10 (primo elemento) numeri[-1] # -> 50 (ultimo elemento) numeri[1:3] # -> [20, 30] (slicing: da indice 1 incluso a 3 escluso)
Schema visivo degli indici:
lista = [a, b, c, d, e] 0 1 2 3 4 # indici positivi -5 -4 -3 -2 -1 # indici negativi

1 Caratteristiche Principali

Definizione

Le liste in Python si definiscono utilizzando le parentesi quadre [ ].

Esempio:

lista = [1, 2, 3]

A differenza di altri linguaggi (come il C/C++, dove si utilizzano gli array monodimensionali o le matrici bidimensionali con vincoli rigidi), le liste in Python sono molto più flessibili e semplici da usare.

Tipi di dato misti

All'interno di una singola lista possono coesistere elementi di tipo diverso:

  • interi → 5
  • stringhe → "ciao"
  • booleani → True
  • float → 3.7

Esempio:

lista_mista = ["mela", 12, True, 4.5]
In C/C++ questo sarebbe impossibile, perché un array deve contenere tutti elementi dello stesso tipo.

Dinamicità

Non c'è bisogno di dichiarare a priori la grandezza della lista (quanti elementi avrà). Le liste sono dinamiche, cioè:

  • si possono aggiungere elementi
  • si possono togliere elementi
  • si possono modificare elementi

Esempio:

lista = [] lista.append(10) lista.append("ciao")

Ordinamento e Indici

Le liste sono insiemi ordinati, cioè ogni elemento ha una posizione precisa. Gli indici partono da 0.

Esempio:

lista = ["a", "b", "c", "d"]
ElementoIndice
"a"0
"b"1
"c"2
"d"3

L’ultimo elemento ha indice dimensione - 1.

Supportano gli indici negativi

Gli indici negativi permettono di partire dalla fine:

  • -1 → ultimo elemento
  • -2 → penultimo
  • -3 → terzultimo

Esempio:

lista = ["mela", "pera", "banana"] print(lista[-1]) # stampa "banana"
2 Creare e Stampare le Liste

Definire una lista vuota e una lista con elementi:

# Definire una lista vuota lista1 = [] # Definire una lista con elementi misti lista2 = ["mela", "ciao", 3.7, 15] # Stampa a video print("lista1:", lista1, "lista2:", lista2)
🔵 Nota di confronto: In Python, la funzione print() stampa direttamente l'intero contenuto della lista. In C o C++ invece, per stampare una lista (array), devi usare un ciclo for e stampare un elemento alla volta.
3 Dimensione delle Liste (Funzione len)

Per sapere quanto è lunga una lista (cioè quanti elementi contiene), si usa la funzione len().

Esempio:

lunghezza1 = len(lista1) lunghezza2 = len(lista2) print("dimensione lista1 =", lunghezza1) print("dimensione lista2 =", lunghezza2)

Per lista2 = ["mela", "ciao", 3.7, 15], la funzione len restituisce 4.

4 Aggiungere Elementi alla Lista

A. Inserimento in coda (append)

Il metodo append() aggiunge sempre l'elemento alla fine della lista.

print("Aggiungi elementi in coda") lista2.append("3AI") print("lista2 =", lista2)

Risultato:

["mela", "ciao", 3.7, 15, "3AI"]

B. Inserimento in una posizione specifica (insert)

Il metodo insert(indice, valore) inserisce un elemento in una posizione precisa, spostando gli altri in avanti.

print("Aggiungi elementi in un posto preciso") # Inserisce il numero 24 nella posizione 2 (cioè dopo "ciao") lista2.insert(2, 24) print("lista2 =", lista2)

Risultato (prima dell’append):

["mela", "ciao", 24, 3.7, 15]
5 Scorrere la Lista con un Ciclo FOR

Se hai bisogno di scorrere la lista e agire su un elemento alla volta (ad esempio per stamparli singolarmente), puoi usare un ciclo for combinato con range() e len().

# Stampa un elemento alla volta for x in range(len(lista2)): print(lista2[x])
🔵 Nota importante: range(len(lista2)) è un’abbreviazione di range(0, len(lista2)). Significa:
  • parti da 0
  • arriva fino a len(lista2)
  • ma ti fermi prima dell’ultimo numero (come sempre nel range)
🔵 Aggiunta utile: scorrere la lista SENZA indici
Python permette anche un modo più semplice:
for elemento in lista2: print(elemento)
6 Metodi e Funzioni per le Liste

La regola d'oro per utilizzare un metodo in Python segue questa sintassi generale:

NomeVariabile.Metodo(argomenti)

Panoramica dei Metodi Principali (Modificano la lista originale)

  • append(x): Aggiunge un elemento alla fine della lista.
  • extend(lista): Estende la lista aggiungendo tutti gli elementi di un'altra lista.
  • insert(i, x): Inserisce un elemento alla posizione specificata i.
  • remove(x): Rimuove la prima occorrenza dell'elemento x.
  • pop([i]): Rimuove e restituisce l'elemento alla posizione i (o l'ultimo se non specificato).
  • sort(): Ordina gli elementi della lista.
  • reverse(): Inverte l'ordine degli elementi della lista.
  • clear(): Rimuove tutti gli elementi dalla lista.

Funzioni Integrate Comuni (Lavorano sulla lista)

  • len(lista): Restituisce il numero di elementi presenti.
  • sorted(lista): Restituisce una nuova lista ordinata, lasciando intatta l'originale.
  • count(x): Conta quante volte il valore x appare nella lista.
  • index(x): Restituisce l'indice della prima occorrenza di x.

Esempi Pratici sui Metodi

# Creazione frutta = ['mela', 'banana', 'mela'] # Aggiungere frutta.append('arancia') # ['mela', 'banana', 'mela', 'arancia'] # Contare conteggio = frutta.count('mela') # Restituisce 2 # Ordinare frutta.sort() # ['arancia', 'banana', 'mela', 'mela']

Esempi Pratici sulla Rimozione Elementi

# Partiamo da una lista esistente lista2 = ["mela", "ciao", 24, 3.7, 15, "3ai"] print(lista2) print("Cancella la prima occorrenza di un certo valore:") lista2.remove(15) print(lista2) print("Rimuovi l'elemento in una certa posizione:") elemento = lista2.pop(0) print(elemento) # Stampa "mela" print("Svuota la lista:") lista2.clear() print("lista2 =", lista2)
7 Liste Annidate (Liste di Liste)

Un concetto avanzato e molto utile è la gestione di liste che contengono altre liste come elementi.

Prendiamo in esame questa lista complessa:

lista = ["Ciao", [1, 2, 3], ["abc", "def"]]

Analisi degli elementi

Ricordando che sono liste solo gli elementi racchiusi tra [ ], notiamo che non tutti gli elementi di questa lista principale sono a loro volta delle liste:

  • lista[0] è la stringa "Ciao" (non è una lista).
  • lista[1] è la lista di numeri [1, 2, 3] (che possiamo chiamare "lista 1").
  • lista[2] è la lista di stringhe ["abc", "def"] (che possiamo chiamare "lista 2").
8 Accedere agli elementi delle "Sottoliste"

Per accedere a un elemento specifico che si trova dentro una lista annidata, devi usare una doppia parentesi quadra.

  • La prima parentesi quadra indica l'indice della lista principale.
  • La seconda parentesi quadra indica l'indice all'interno della sottolista.

Se vogliamo accedere ai numeri della prima sottolista:

# lista[1] prende l'intera lista [1, 2, 3] # Per stampare i singoli numeri al suo interno: print(lista[1][0], lista[1][1], lista[1][2]) # Risultato a schermo: 1 2 3
9 Operazioni Pratiche: I Cicli FOR sulle Sottoliste

È possibile usare i cicli for per compiere operazioni specifiche sugli elementi interni a una lista complessa.

A. Sommare gli elementi di una sottolista numerica

Per sommare i numeri 1, 2, 3 contenuti in lista[1], si usa un ciclo. Nota bene: in Python, per fare una somma cumulativa, devi prima dichiarare una variabile di partenza uguale a zero.

somma = 0 # Variabile di appoggio iniziale for i in lista[1]: # Cicla attraverso i numeri [1, 2, 3] somma += i # Equivalente a: somma = somma + i print("La somma è:", somma) # Risultato: 6

B. Ciclare su una stringa (come se fosse una lista)

Se eseguiamo un ciclo direttamente su un elemento stringa, ad esempio scrivendo for j in lista[0]: print(j), Python leggerà la parola carattere per carattere. Dato che lista[0] è la parola "Ciao", questo ciclo stamperà ogni lettera verticalmente:

for j in lista[0]: print(j)

Risultato:

C i a o
10 Controllo dei Tipi degli Elementi

Essendo le liste flessibili, possiamo inserire dati di natura diversa. Python ci fornisce metodi pratici per controllare il tipo di ogni elemento, come la funzione type() o la funzione isinstance().

l = ["ciao", 23, [1, 2, 3]] # Visualizza i tipi degli elementi di una lista for x in l: print(type(x)) print("-----------------") # Controlla in modo specifico se un elemento appartiene a una categoria for x in l: if isinstance(x, list): print(x, "è una lista") elif isinstance(x, (float, int)): print(x, "è un numero") elif isinstance(x, str): print(x, "è una stringa")

Analisi del Codice

  • type(x): Questa funzione restituisce il tipo di dato esatto di una variabile (es. <class 'int'> per un intero, <class 'str'> per una stringa). È utile quando vogliamo semplicemente stampare a schermo o ispezionare "di che pasta è fatto" un elemento.
  • isinstance(x, tipo): Questa è la funzione più potente ed usata per i controlli nei programmi reali. Prende due parametri: l'elemento da analizzare e il tipo da verificare (es. list, int, str). Restituisce True se l'elemento appartiene a quel tipo, e False altrimenti.
  • isinstance(x, (float, int)): Notiamo che al posto di un singolo tipo, abbiamo passato due tipi tra parentesi (float, int). Questa è una comodità di Python: permette di chiedere "questo elemento è un float OPPURE un intero?" in una sola riga di codice. Molto comodo per verificare se qualcosa è genericamente un "numero".

📘 SPIEGAZIONI TEORICHE (semplici e concrete)

🔵 LISTE

  • Si creano con le parentesi quadre: [ ]
  • Sono dinamiche: si possono riempire, svuotare, modificare
  • Possono contenere numeri, stringhe, ecc.

🔵 FOR + RANGE

Il ciclo:

for i in range(start, stop, step):

significa:

  • start → da dove parto
  • stop → dove mi fermo (mi fermo prima di questo numero)
  • step → di quanto avanzo ogni volta

Esempio:

range(1, 101, 1)

→ numeri da 1 a 100.

🔵 APPEND

Serve per aggiungere un elemento alla lista:

lista.append(x)

🔵 DIVISIBILITÀ

Un numero è divisibile per 5 se:

x % 5 == 0

Se il resto è diverso da zero → non è divisibile.

🔵 NUMERI PARI

Un numero è pari se:

x % 2 == 0
E2 Ricerca di un Elemento e Indice
🎯 Testo del Problema:

Scrivi un programma che a partire da un elemento e una lista di elementi dica in output se l'elemento passato sia presente o meno. Qualora l'elemento sia presente nella lista, il programma dovrà comunicarci l'indice dell'elemento tramite il metodo index().

✅ VERSIONE 1 — Ricerca con il ciclo FOR + flag "trovato"

# Lista di numeri già pronta lista1 = [2, 5, 12, 8, 16, 52, 500, 1000, 65, 9] # Calcolo la lunghezza della lista usando len() # len() restituisce quanti elementi ci sono nella lista n = len(lista1) # Variabile che useremo come "bandierina" per sapere se l'elemento è stato trovato trovato = False # Chiedo all'utente quale numero vuole cercare # input() legge un testo, int() lo trasforma in numero intero num = int(input("Inserisci l'elemento da controllare --> ")) # Ciclo for per scorrere tutta la lista elemento per elemento for x in lista1: # Controllo se l'elemento corrente è uguale al numero cercato if x == num: # Se sono uguali, allora l'elemento esiste nella lista print("L'elemento esiste") # Uso il metodo index() per sapere in quale posizione si trova indice = lista1.index(num) print("In posizione:", indice) # Segno che l'elemento è stato trovato trovato = True # Dopo il ciclo, se trovato è ancora False, significa che non abbiamo trovato nulla if trovato == False: print("Elemento non trovato")

📘 SPIEGAZIONE TEORICA (semplice)

🔵 Il ciclo FOR

Serve per scorrere una lista elemento per elemento:

for x in lista1:

Significa: prendi il primo elemento della lista e chiamalo x, poi il secondo, poi il terzo... fino alla fine.

🔵 Il metodo index()

lista1.index(num) restituisce l'indice (cioè la posizione) del numero cercato. Ricorda: gli indici partono da 0.

🔵 La variabile "trovato"

Serve come "bandierina" logica:

  • False → non trovato
  • True → trovato

Alla fine del programma controlliamo questa bandierina. Se non si è mai alzata (if trovato == False:), significa che l'elemento non c'era.

✅ VERSIONE 2 — Ricerca con l’operatore IN (più semplice ed efficiente)

# Lista di numeri lista1 = [2, 5, 12, 8, 16, 52, 500, 1000, 65, 9] # Chiedo il numero da cercare num = int(input("Inserisci l'elemento da controllare --> ")) # Controllo direttamente se num è dentro la lista # L'operatore in restituisce True se l'elemento è presente if num in lista1: print("L'elemento esiste") # index() restituisce la posizione dell'elemento indice = lista1.index(num) print("In posizione:", indice) else: print("L'elemento non è presente nella lista")

📘 SPIEGAZIONE TEORICA DEL METODO "IN"

L’operatore speciale di Python:

if num in lista1:

Significa esplicitamente: "controlla se num è dentro lista1". È il modo più semplice, leggibile e veloce per cercare un elemento all'interno di una lista in Python, senza dover per forza usare cicli o variabili bandierina.

E3 Elementi in Posizione Pari
🎯 Testo del Problema:

Data una lista di n interi, con n valore in input, creare un'altra lista contenente solo gli elementi di posizione pari della prima lista.

🧠 Ragionamento Semplice

  • L'utente decide quanti numeri inserire → n.
  • Riempio una lista con n numeri.
  • Creo una seconda lista.
  • Scorro la lista con l'indice i.
  • Se l'indice i è pari (i % 2 == 0), prendo quell'elemento e lo metto nella seconda lista.
  • Alla fine stampo la lista originale e la lista con gli elementi in posizione pari.

💻 Codice Commentato e Spiegato

# Chiedo all'utente quanti numeri vuole inserire n = int(input("Quanti numeri vuoi inserire? ")) # Creo una lista vuota che conterrà i numeri inseriti lista1 = [] # Uso un ciclo for per inserire n numeri # range(0, n, 1) significa: # start = 0 (parto da 0) # stop = n (mi fermo prima di n) # step = 1 (avanzo di uno alla volta) for i in range(0, n, 1): x = int(input("Inserisci un numero: ")) lista1.append(x) # Creo una seconda lista vuota che conterrà solo gli elementi in posizione pari lista_pos_pari = [] # Scorro di nuovo la lista usando l'indice for i in range(0, n, 1): # Controllo se l'indice è pari # Un numero è pari se il resto della divisione per 2 è zero if i % 2 == 0: # Se l'indice è pari, prendo l'elemento in quella posizione elemento = lista1[i] # Lo aggiungo alla seconda lista lista_pos_pari.append(elemento) # Stampo le due liste print("Lista originale:", lista1) print("Elementi in posizione pari:", lista_pos_pari)
E4 Elementi di Valore Dispari
🎯 Testo del Problema:

Data una lista di n interi, con n valore in input, creare un'altra lista contenente solo gli elementi di valore dispari della prima lista.

🧠 Ragionamento Semplice

  • L'utente decide quanti numeri inserire.
  • Riempio la lista con questi numeri.
  • Creo una seconda lista.
  • Scorro la lista elemento per elemento.
  • Se il numero è dispari (x % 2 != 0), lo metto nella seconda lista.
  • Alla fine stampo la lista originale e la lista con i numeri dispari.

💻 Codice Commentato e Spiegato

# Chiedo quanti numeri vuole inserire l'utente n = int(input("Quanti numeri vuoi inserire? ")) # Creo la lista vuota lista1 = [] # Riempio la lista con n numeri for i in range(0, n, 1): x = int(input("Inserisci un numero: ")) lista1.append(x) # Creo una seconda lista per contenere solo i numeri dispari lista_dispari = [] # Scorro la lista elemento per elemento for x in lista1: # Un numero è dispari se il resto della divisione per 2 NON è zero if x % 2 != 0: # Se è dispari, lo aggiungo alla lista dei dispari lista_dispari.append(x) # Stampo le due liste print("Lista originale:", lista1) print("Elementi di valore dispari:", lista_dispari)
E5 Ricerca Sottostringa senza Metodi Nativi
🎯 Testo del Problema:

Scrivere una funzione cerca(testo, parola) che, data una stringa testo e una stringa parola, restituisce la posizione iniziale della prima occorrenza di parola in testo. Se non c'è alcuna occorrenza restituisce -1. Vietato l'uso di funzioni di libreria come .find() o .index().

🧠 Ragionamento Semplice

  • Devo cercare una parola dentro un testo.
  • Scorro il testo lettera per lettera (come una "finestra" che scorre).
  • In ogni posizione i del testo, controllo se da lì in poi le lettere coincidono con la parola.
  • Uso un secondo indice j per scorrere le singole lettere della parola.
  • Se tutte le lettere coincidono → ho trovato la parola → restituisco la posizione i.
  • Se arrivo alla fine del testo senza trovare nulla → restituisco -1.

🚫 Che significa "non usare funzioni di libreria"?

Significa NON usare funzioni o metodi già pronti di Python che risolvono il problema al posto nostro. In questi esercizi, l'obiettivo è imparare a costruire la logica algoritmica "da zero", usando unicamente i cicli (for / while) e le condizioni (if / else).

In pratica, per questo esercizio NON puoi usare strumenti comodissimi come:

Cosa è vietatoCosa fa in automatico
.index(x) Trova subito la posizione di un elemento.
Es: lista.index(5)
.find(x) Cerca una sottostringa e ne restituisce l'indice.
Es: "ciao".find("a")
in (per cercare) Operatore logico che dice istantaneamente se un elemento esiste.
Es: if "a" in "ciao":
.count(x) Conta automaticamente quante volte appare un elemento.
Es: lista.count(5)
.split() Spezza una frase trasformandola in una lista di singole parole.
Es: frase.split()
.replace(x, y) Sostituisce un testo con un altro in un solo colpo.
Es: frase.replace("a", "b")
max() e min() Trovano il valore massimo o minimo senza dover fare confronti manuali.
Es: max([1, 5, 2])
sum() Somma tutti i numeri di una lista senza usare una variabile accumulatore.
Es: sum([10, 20])
