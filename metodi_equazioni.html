✅ TIPO A — Elementari: stessa base già presente
Se a destra e a sinistra c'è già la stessa base, cancello le basi e ugualio solo gli esponenti.
Esempio diretto
3x = 3−3→ stessa base (3) →x = −3
💡 Regola d'oro: bA = bB → A = B (cancello le basi e ugualio gli esponenti)
🔧 TIPO B — Riconducibili: 3 operazioni possibili
🔹 1 — Scompongo la base
27x→ 27=3³ →(3³)x = 33x
🔹 2 — Ribalto la frazione (esponente negativo)
3x = 13−3→ ribалто →3x = 33 → x = 3
🔹 3 — Trasformo radice in esponente frazionario
3x = 4√(3²)→3x = 32/4 → x = 1/2
🔢 Quanti termini? — Il metodo cambia!
📗 1 o 2 termini → metodo diretto
Applico le proprietà delle potenze (× ÷ potenza), riconduco alla stessa base, ugualio gli esponenti.
2x · 23 = 25→x+3 = 5 → x = 2
📙 3+ termini con + e − → sostituzione con t
Con + e − non posso usare direttamente le proprietà delle potenze.
Uso il cambio di parametro t: chiamo t l'esponenziale comune → risolvo in t → torno a x.
Esempio 1 — Equazione trinomia
22x − 3 · 2x − 4 = 0
Chiamo t = 2x
1
Sostituisco
22x = (2x)² = t²
2x = t
L'equazione diventa: t² − 3t − 4 = 0
↓
2
Calcolo il Δ (a=1, b=−3, c=−4)
Δ = b² − 4ac = (−3)² − 4·(1)·(−4)
Δ = 9 + 16 = 25

t1,2 = 3 ± √252 = 3 ± 52

t₁ = 3 + 52 = 82 = 4     t₂ = 3 − 52 = −22 = −1
↓
3
Ritorno a x — caso t₁ = 4
t₁ = 4 > 0 ✓

2x = 4 = 2² → stessa base → x = 2
↓
4
Ritorno a x — caso t₂ = −1
t₂ = −1 < 0 🚫

2x = −1 → IMPOSSIBILE
L'esponenziale è sempre positiva → non può mai valere −1 → scarto questa soluzione
🏆 SOLUZIONE ESEMPIO 1
x = 2
Esempio 2 — con frazione (1/3x + 3x = 2)
13x + 3x = 2
Chiamo t = 3x → 13x = 1t
1
Sostituisco
1t + t = 2
↓
2
Moltiplico tutto per t (con t ≠ 0)
1 + t²t = 2tt

1 + t² = 2t
t² − 2t + 1 = 0
↓
3
Calcolo il Δ (a=1, b=−2, c=1)
Δ = b² − 4ac = (−2)² − 4·1·1 = 4 − 4 = 0

t1,2 = 2 ± √02 = 22 = 1  (soluzione doppia)
↓
4
Ritorno a x — caso t = 1
t = 1 > 0 ✓

3x = 1 = 3⁰ → stessa base → x = 0

⚠ Ma aspetta: verifichiamo nel testo originale!
13⁰ + 3⁰ = 1 + 1 = 2 ✓ → la soluzione soddisfa l'equazione

Dagli appunti: t=1 → 3x=1 → 3x=3⁰ → x=0 SOLUZIONE IMPOSSIBILE
(perché x=0 rende il denominatore 3⁰=1 ≠ 0, quindi è accettabile matematicamente — va verificato con il testo originale)
🚫 Dagli appunti del professore: x = 0 — Soluzione impossibile
Verificare sempre anche le condizioni di esistenza (denominatore ≠ 0 e t > 0)
🚫 Attenzione: Le proprietà delle potenze valgono solo con × e ÷. Con + e − non posso semplificare!
Es: 2x + 2x ≠ 22x  ·  Corretto: 2x + 2x = 2 · 2x
🗺️ Schema riassuntivo — Come procedere
1
Conto i termini (separati da + − o =)
2
Riconduco alla stessa base → scompongo / ribalto / trasformo radici
3
1–2 termini? → proprietà potenze → ugualio esponenti
3+ termini con +/−? → sostituzione t → risolvo in t → torno a x
4
Verifico sempre: t > 0 (esponenziale sempre positiva!) e denominatore ≠ 0

Stessa base → elementare · Basi diverse → riconduci · 1–2 termini → proprietà potenze · 3+ con +/− → sostituzione t · t sempre > 0 · verificare sempre!
